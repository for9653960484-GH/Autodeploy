# Простое Docker-приложение на Python для теста

## Что делает
Небольшой HTTP-сервер, который отвечает простым текстовым сообщением и
предоставляет два эндпоинта для простых вычислений: `/summ` и `/mult`.
Полезен для проверки, что Docker-контейнер запускается и принимает запросы.

## Сборка образа
```bash
docker build -t python-docker-test .
```

## Запуск контейнера
```bash
docker run --rm -p 8000:8000 python-docker-test
```

## Проверка работы
1. Откройте http://localhost:8000/ в браузере — это проверит, что сервер жив.
2. Проверьте сложение: http://localhost:8000/summ?a=2&b=3
3. Проверьте умножение: http://localhost:8000/mult?a=2&b=3

## Опционально: изменить порт
Измените значение переменной `PORT` и проброс порта после `-p`.
Например, чтобы использовать порт 5000:
```bash
docker run --rm -e PORT=5000 -p 5000:5000 python-docker-test
```
