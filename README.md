# Простое Docker-приложение на Python для теста

## Что делает
Небольшой HTTP-сервер на Python, который отвечает простым текстовым сообщением
и предоставляет два эндпоинта для простых вычислений: `/summ` и `/mult`.
Также есть простой фронтенд (HTML + JS), который обращается к бэкенду через
внутренний прокси в `docker-compose`.

## Сборка образа
```bash
docker build -t python-docker-test .
```

## Запуск контейнера
```bash
docker run --rm -p 8000:8000 python-docker-test
```

## Docker Compose (бэкенд + фронтенд)
1. Откройте `docker-compose.yml` и замените образ бэкенда на ваш
   (сейчас указан `flask-backend:latest`).
2. Запустите compose:
```bash
docker compose up -d
```
3. Откройте фронтенд: http://localhost:8080/
4. Нажмите «Сложить» или «Умножить» и посмотрите результат.

## Проверка работы (без фронтенда)
1. Откройте http://localhost:8000/ в браузере — это проверит, что сервер жив.
2. Проверьте сложение: http://localhost:8000/summ?a=2&b=3
3. Проверьте умножение: http://localhost:8000/mult?a=2&b=3

## Опционально: изменить порт
Измените значение переменной `PORT` и проброс порта после `-p`.
Например, чтобы использовать порт 5000:
```bash
docker run --rm -e PORT=5000 -p 5000:5000 python-docker-test
```
